Editext++

A text editor developed as Windows Form Application for HTML-CSS editing , basic formatting and password encription techniques. Therefore, once you have written a file, one can secure the file with a password with data encrypted.

Welcome to Editext Page. Use this page to know more about editext and understand the features and usability of this simple interface text editor developed using VC++ and Windows Form Application

Using Editext

$ git clone https://github.com/priyankt68/Editext.git

Designer Templates

Editext supports following features as a text editor:

  • Formatting features,viz-a-viz, cut, copy , paste, font formatting, bold,italics,underline.
  • File options such as Save, Save As, Open the source folder.
  • Password Protection of file. File cannot be accessed without the password set by the author. Even if one is able to view the file, the data is encrypted.
